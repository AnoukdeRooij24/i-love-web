3. use green hosting
    - the cloud is real!

    - there is only one grid

4. only necessary functions
    - in het minste geval gaat het gebruik 2x maar waarschijnlijk 4x

    - opt-out is een goede optie voor minder carbon uitstoot
    - "work on removing rather than adding"
