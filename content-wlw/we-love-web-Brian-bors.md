### Social modal of Accessibility
- "Je kunt hier op de invalide plek parkeren, alles met minder dan 8 ledematen is een disababeld persoon hier" Rick & Morty
- Daar waar wij beperkt mee raken is niet de medische reactie maar het verschil tussen het design en gebruiker
- Beperking komt voor uit de sociale omgeving
- Vooral websites (alle digitale omgevingen)
- "curb cut" effect -> voor een doelgroep wordt iets ontworpen wat voor andere doelgroeppen ook handig is
- Ontwerp iets voor iemand met een beperking en dan help je daar ook andere mee die je niet bewust zal bedenken

### WCAG texten lezen
- Je kan de tekst opdelen in 3 onderdelen met kleurcodes en nummers (bijv. 1) ) gebruiken om het leesbaarder te maken
    1) Relevante situatie
    2) Eis
    3) Uitzonderingen
- WCAG is technologie agnostisch geschreven (kan voor heel veel verschillende dingen en technieken gebruikt worden)
- WCAG criteria bestaan uit 3 nummers bijv. 1.2.3.
- 1e nummer is princiepe 
    - percievable -> output
    - Operable -> input
    - understandeble -> output moet ook begrepen worden
    - Robust -> compatible met hulpsoftware
- 2e nummer is richtlijn 
- 3e nummer is succescriteria

## Criteria 1.3.1
- Meest afgekeurd
- Percievable
- Als er een relatie met een verschil te zien is op een website moet dit in de code ook duidelijk zijn
- Over het algemeen kan je dit voor elkaar krijgen door goeie HTML te schrijven, dan leg je de connecties automatisch zonder CSS te gebruiken
