## Design stystem
- Een centrale plaats waar je je design van je website regeld en code vast steld om een constistente user experience
- Vitamin play (van decathlon)
- Experiences to move people through the wonder of sport (slogan)
- A suite of guidelines, reusable components, modules en template including
- Foundation in Figma:
    - colors
    - typografy
    - spacing (ook responsive grid)
    - border radius
- foundations worden omgezet in design tokens 
- desgin tokens komen ook weer in figma terug, JSON file export en die type naam geef je een value en die value geef je mee in CSS variable
- Design en enginering spreken op deze manier de zelfde taal
- design stystem zit ook icons, assets en email look and feel
- Een look en feel over alle channels
- Waarom een design system: 
    - consistentie over alle kanalen en producten
    - efficient, snel paginas opzetten met vaste componenten
    - schaalbaar 
    - verbeterde samenwerking tussen design en IT
    - betere kwaliteit
    - accessibility
    - onboarding en training voor nieuwe werknemers
- Wat niet in een design system?
    - page specific of flow specific UI
    - Complete pages
    - Highly couples to backend 
    - One off's
    - Experimenten
    - specific things to one product

## Vitamin play
- github repo (niet openbaar)
- Daar zit alle code in: componenten, documentatie en design tokens
- Frameworks: React, Svelte & Vue (not recommended)
- Is een npm package
- Wanneer zet je iets in een design system? Als het over meerdere platforms wordt verdeeld
- Zorg dat ze composeble zijn 
- Zorg dat je components:
    - Composeble
    - Flexible
    - Accessible
    - Themed (light/dark mode)
    - Well documented
    - Consistent
    - Stable en predicteble
- Developer portal (guidelines hoe het werkt en welke properties gerbuikt worden en hoe de accessibility gebruikt wordt)
- Vision simulator in de test fase
