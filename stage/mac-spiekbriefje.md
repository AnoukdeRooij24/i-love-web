# Spiekbriefje mac shortcuts

## Mac

|  |  |
| --- | --- |
| Venster wisselen | ⌘ + tab |
| Emoji | ⌘ + option + spatie |
| Zoeken naar apps | ⌘ + spatie |
| Full screen screenshot | ⌘ + shift + 3 |
| Deel van screen screenshot | ⌘ + shift + 4 |
| App sluiten | ⌘ + Q |
| Video opnemen | ⌘ + shift + 5 |
| Rechtermuis klik | Twee vingers midden mousepad / control click |
| Delete | fn + ⌫ |
| Trema | Option + u (daarna letter die je wilt met trema) |

## In apps

|  |  |
| --- | --- |
| Inspect google | ⌘ + option + i |
| Zoeken in google (pagina) | ⌘ + F |
| Tokens studio in figma | ⌘ + P |
| iTerm openen | ⌘ + ` (backtick) |
| Instellingen openen in app | ⌘ + , (komma) |
| Inspect in polypane | Option + hover / shift + ⌘ + C |
| Polypane breakpoints | ⌘ + B |
| Polypane all windows screenshot | ⌘ + P |
| Command bar | ⌘ + K |
| Gesloten tab heropenen google | ⌘ + shift + T |
